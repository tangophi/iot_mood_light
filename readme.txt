This repo contains code for an Arduino/ESP8266 based IoT Mood Light project.

Details of the project can be found at:
http://www.instructables.com/id/IoT-Mood-Lighting/

room_light_arduino contains the Arduino sketch for an Arduino Pro Mini.  Use Arduino IDE to flash the code to the microcontroller.

room_light_esp8266 contains the firmware source for ESP8266.  Use the ESP8266 SDK kit to compile and flash the firmware to the module.

room_light_kicad contains the kicad files for the circuit used in the project.

images contains photos of the project during various stages of construction and installation.
